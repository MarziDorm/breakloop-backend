# BreakLoop

A habit-breaking app that helps you identify triggers, log urges, and learn from slip-ups.

## How to run

**Backend** (Terminal 1)
```
cd breakloop-main/backend
npm install
npm install cors
node index.js
```

**Frontend** (Terminal 2)
```
cd breakloop-main/frontend
npm install
npm start
```

The app runs at `http://localhost:3000` (backend) and `http://localhost:3001` (frontend).
